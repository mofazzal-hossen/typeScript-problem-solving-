# Assignment 1 - Advanced Problem Solving with TypeScript & OOP

## 📌 Project Overview

This assignment focuses on solving different TypeScript programming problems using strong typing, generics, interfaces, classes, inheritance, and object-oriented programming concepts.

The goal of this project is to practice writing clean, reusable, and type-safe TypeScript code while improving problem-solving skills.
